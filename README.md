# 🛒 Amazon Clone

A fully responsive **Amazon Clone** built using **React.js**, **CSS**, and **JavaScript**, replicating the core UI/UX features of the real Amazon website.

---

## 📌 Features

- 🏠 **Homepage** similar to Amazon
- 🔍 **Search bar** to find products
- 🛍 **Product listing** with images, prices, and ratings
- 🛒 **Add to Cart** functionality
- 🧾 **Cart page** showing selected products and total price
- 📱 Fully **responsive** design for mobile, tablet, and desktop

---

## 🖥 Tech Stack

- **Frontend:** React.js, JavaScript, HTML5, CSS3
- **Icons & Styling:** CSS Flexbox, Grid, FontAwesome
- **Package Manager:** npm
